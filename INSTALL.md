# 📥 Installation and Startup Guide

Follow these steps to deploy the DARWIN telemetry stack on your local machine or Raspberry Pi.

## 📋 Prerequisites
* **OS:** Linux (Ubuntu 22.04+ recommended) or WSL2.
* **Kernel:** 5.8+ (Required for eBPF features).
* **Tools:** Docker Desktop (or Docker Engine), Python 3.10+, and `uv` (recommended).
